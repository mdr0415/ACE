# ACE 플랫폼 - 외국인 유학생 통합 지원 시스템

## 📋 프로젝트 개요

ACE (ACADEMICS · COMMUNITY · EMPLOYMENT) 플랫폼은 외국인 유학생들의 학업, 생활, 취업을 전주기적으로 지원하기 위해 설계된 통합 웹 기반 지원 시스템입니다.

### 🎯 프로젝트 목표

- 외국인 유학생의 한국 대학 생활 적응 지원
- 학업 능력 진단 및 맞춤형 교육 추천
- 전문가 상담 및 AI 기반 학사 안내 제공
- 지역 취업 정보 및 비자 지원 혜택 제공
- 선배들의 취업 성공 사례 공유 및 멘토링

## ✨ 주요 기능

### 🎓 학업 지원 (A-Academics)

#### 1. 한국어 레벨 진단 ✅
- **3가지 테스트 옵션:**
  - TOPIK 공식 기출문제
  - 세종학당 레벨테스트
  - 누리-세종학당 온라인 테스트
- 결과 화면 업로드 (이미지 파일 선택 / 클립보드 붙여넣기 지원)
- 자동 레벨 분석 및 점수 저장
- 학습 성장 타임라인 시각화 (Chart.js)
- 다음 목표까지의 진행률 표시
- 맞춤형 강좌 추천

**상태**: ✅ 구현 완료

#### 2. 학교생활 안내 ✅
- 학사 규정 안내
- 수강신청 일정 및 방법
- 장학금 가이드라인
- 졸업 이수 요건

**상태**: ✅ 구현 완료 (학사 정보 안내)

### 🏠 생활 지원 (C-Community)

#### 1. 전문가 상담 ✅
- 카테고리별 상담 신청 (학업/생활/진로/정서/행정)
- 상담 내역 조회 기능
- 답변 상태 추적 (접수됨/확인중/답변완료)

**상태**: ✅ UI 구현 완료

#### 2. 한국생활 안내 ✅
- 주거 정보 (기숙사 및 자취)
- 의료 서비스 및 보험 안내
- 유학생 관련 법률 정보

**상태**: ✅ 구현 완료 (주거·의료·법률)

#### 3. AI 챗봇 ✅
- 자주 묻는 학사 질문에 대한 즉각적인 답변
- 자연어 기반 대화형 상담
- 태그를 이용한 빠른 질문 선택
- 실시간 채팅 인터페이스

**상태**: ✅ 구현 완료

### 💼 진로 지원 (E-Employment)

#### 1. 실시간 채용정보 ✅
- 인기 키워드 태그 제공
- 비자 지원 및 직무별 샘플 채용 카드
- 실시간 업데이트를 고려한 UI 설계

**상태**: ✅ 구현 완료

#### 2. 지역 취업정보 ✅
- 지역/업종별 필터링 기능 (익산, 전주, 전북 등)
- 외국인 채용 가능 기업 우선 표시
- 비자 스폰서 가능 기업 필터링
- 상세 채용 정보 (급여, 위치, 복리후생)

**상태**: ✅ 구현 완료 (학교가족회사 중심)

#### 3. F-2-R 비자 점수 계산기 ✅
- 나이, 학력, TOPIK 레벨 기반 점수 산출
- 실시간 점수 계산 및 자격 요건 판정
- 맞춤형 점수 향상 방법 제안
- 시각적인 점수 표시 (원형 차트)

**상태**: ✅ 구현 완료

#### 4. 선배 취업 스토리 ✅
- 상세한 취업 성공 사례 소개
- 타임라인 기반 준비 과정 공유
- 면접 경험담 및 Q&A 제공
- 비자 전환 과정 안내
- 멘토링 신청 기능

**상태**: ✅ 구현 완료

## 🎨 3단계 메뉴 시스템

### 네비게이션 구조:
- **Depth 1 (Primary)**: 메인 - 홈 버튼
- **Depth 2 (Main Features)**: 한국어 진단, 전문가 상담, 취업정보, 선배 스토리, 비자 계산기
- **Depth 3 (Sub Pages)**: 진단 결과, AI 상담

**주요 특징**:
- 색상 구분을 통한 시각적 계층 구조 형성
- 명확한 이동 경로 제공
- 모바일 대응 반응형 디자인
- **헤더 뒤로가기 탐색**: '←' 기호가 있는 모든 화면의 헤더를 클릭하여 메인 화면으로 복귀 가능

## 🌐 다국어 지원

### 지원 언어 (7개국어):
- 🇰🇷 **한국어**
- 🇺🇸 **English**
- 🇺🇿 **Uzbek** (O'zbek)
- 🇨🇳 **Chinese** (中文)
- 🇻🇳 **Vietnamese** (Tiếng Việt)
- 🇲🇳 **Mongolian** (Монгол)
- 🇱🇰 **Sinhala** (සිංහල)

### 번역 시스템:
- 통합 번역 파일 관리 (`js/translations.js`)
- 동적 언어 전환 및 LocalStorage 설정 저장
- 국기 이모지를 포함한 세련된 언어 선택 모달
- `data-i18n` 속성을 이용한 모든 UI 요소의 다국어 대응

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그라데이션, 애니메이션
- **JavaScript (ES6+)**: 동적 인터랙션 및 로직
- **Chart.js**: 학습 성장 그래프 시각화
- **Google Fonts**: Noto Sans KR

### 디자인 테마
- **클린 화이트 배경**: 현대적이고 깔끔한 화이트 테마
- **프라이머리 그라데이션**: #1e3c72 → #2a5298 적용
- **모바일 퍼스트**: iPhone X 프레임 기반 설계 (375px × 812px)
